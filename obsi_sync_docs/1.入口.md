# 项目说明
用 wails3 + svelte5 实现一个桌面端app, 用于同步各个obsidian库的配置

### 项目架构
Go = 业务核心 / 同步引擎
Svelte = 操作面板 / 可视化界面
Wails = 两边通信桥
