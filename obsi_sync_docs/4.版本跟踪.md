# 版本跟踪

| 版本 | 阶段 | 需求 | 后端接口 | 核心规则 | 状态 |
| --- | --- | --- | --- | --- | --- |
| M1 | 扫描可用 | 选择目录；扫描出 vault；展示列表 | `ScanVaults(root string) ([]VaultInfo, error)` | 目录下存在 `.obsidian/` 即认为是 Obsidian Vault；跳过隐藏/系统目录、`node_modules`、`.git`；限制最大递归深度 5~8 层 | 待开发 |

## M1 说明

返回结果给前端展示。建议返回字段：

```go
type VaultInfo struct {
	Path string `json:"path"`
	Name string `json:"name"`
}
```

发现某个目录已经是 vault 后，默认不继续深入该 vault 内部扫描。
